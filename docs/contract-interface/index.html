Contract-Interface

Wrapper object of web3.js

Installation

$ cd your_project
$ cd node_modules
$ git clone git@github.com:RealStake/contract-interface.git
$ cd contract-interface
$ npm install

Dependencies

npm i -g webpack webpack-cli

Start development

Open terminal then execute:

webpack-cli

In another terminal execute:

node dev.js

Open browser: http://localhost:8080/public/index.html
